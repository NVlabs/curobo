Build Robot Model
=================

.. automodule:: curobo.examples.getting_started.build_robot_model
   :no-members:

Once you have run the tutorial, open
:mod:`curobo.examples.getting_started.build_robot_model` in your editor.
The inline comments walk through the key design decisions: how sphere density
affects collision checking fidelity, when to raise or lower the MorphIt loss
weights, and how the self-collision matrix is pruned.
